Introduction¶

NV01:NV40 chipsets have the following clocks:

  • core clock [NVPLL]: used to clock the graphics engine - on NV04+ cards only
  • memory clock [MPLL]: used to clock memory and, before NV04, the graphics engine - not present on IGPs
  • video clock 1 [VPLL]: used to drive first/only video output
  • video clock 2 [VPLL2]: used to drive second video output - only on NV11+ cards
